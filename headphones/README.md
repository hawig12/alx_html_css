
## 💡 Key Features

### ✅ Responsive Design
- Fully responsive layout using CSS media queries
- Switches to **mobile version** when the screen width is **≤ 480px**

### 🎨 Interactions
- **Links:**
  - `hover` / `active` state color: `#FF6565`
- **Buttons:**
  - `hover` / `active` state: `opacity: 0.9`

### 📐 Layout
- Content is **centered**
- Maximum width: `1000px`
